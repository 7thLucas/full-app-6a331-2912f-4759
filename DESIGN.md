# Design Guidelines — Clinic Coordinator

## Color Palette
- Primary: Teal (#0f766e / #14b8a6) — primary actions, active states, highlights
- Background: White (#ffffff) — clean, clinical base
- Text: Slate-900 for headings, Slate-600 for body copy
- Accent surfaces: Teal-50 (#f0fdfa) with Teal-200 border (#99f6e4)
- Status colors: Yellow for waiting, Blue for in-consultation, Green for done, Red for alerts

## Typography
- Headings: Bold/Black weight, tight leading
- Body: Regular/Medium, relaxed leading, generous line height
- Labels/chips: Bold, uppercase, tracked, small size

## Elevation & Cards
- Cards: rounded-2xl, subtle border (border-slate-100 or teal-200), white or teal-50 background
- Shadows: shadow-sm only — restrained

## Components
- Primary button: solid teal-600 bg, white text, rounded-xl
- Status badges: color-coded rounded-full pills
- Navigation: clean sidebar with icon + label; active state in teal
- Forms: clean, well-spaced fields, clear labels, minimal decoration
- Tables/lists: clean rows, hover highlight, clear typography hierarchy

## Aesthetic
- Healthcare-appropriate — functional, organized, calm
- No decorative flourishes; efficiency-first
- Like a well-run clinic: everything in its place, nothing wasted
